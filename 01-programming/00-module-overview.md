# Module 01: Programming

## What This Module Covers
এই মডিউলে আমরা সফটওয়্যার ইঞ্জিনিয়ারিং এর একদম কোর প্রোগ্রামিং কনসেপ্টগুলো নিয়ে আলোচনা করব। এখানে ভেরিয়েবল, কন্ট্রোল ফ্লো থেকে শুরু করে অবজেক্ট অরিয়েন্টেড প্রোগ্রামিং (OOP), ডিজাইন প্যাটার্ন, ডেটাবেস, নেটওয়ার্কিং এর বেসিক এবং API ডিজাইন কভার করা হবে। 

## Why It Matters
WellDev Trainee Software Engineer MCQ টেস্টের জন্য এটি অত্যন্ত গুরুত্বপূর্ণ, কারণ এখানে আপনার বেসিক কোডিং সেন্স, আর্কিটেকচারাল নলেজ এবং বিভিন্ন টেকনোলজির (যেমন- ডাটাবেস, নেটওয়ার্কিং) প্র্যাকটিক্যাল জ্ঞান যাচাই করা হবে। এখানে থিওরির চেয়ে প্র্যাকটিক্যাল অ্যাপ্লিকেশন এবং রিয়েল-লাইফ সিনারিও-ভিত্তিক প্রশ্ন বেশি থাকবে।
