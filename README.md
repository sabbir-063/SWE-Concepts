# Software Engineering (SWE) Learning Projects

Welcome to the Software Engineering learning repository! This project is dedicated to learning and practicing core Software Engineering concepts step by step, including **Design Patterns**, **SOLID Principles**, and more. 

As new concepts are learned, this repository will be continuously updated with new implementations and notes.

## 📂 Current Contents

### 1. Design Patterns (`/DesignPatterns`)
Contains Java implementations of the Gang of Four (GoF) design patterns. The patterns are categorized into three main folders:
- **Behavioral/**: Patterns that identify common communication patterns between objects and realize these patterns.
- **Creational/**: Patterns that deal with object creation mechanisms, trying to create objects in a manner suitable to the situation.
- **Structural/**: Patterns that ease the design by identifying a simple way to realize relationships between entities.

*Note: All Java classes have been updated to reflect their new package structure (`DesignPatterns.Category.Pattern`).*

### 2. Design Patterns Guide (`design-patterns.md`)
A comprehensive guide to all 23 GoF Design Patterns written in a mix of English and Bengali. This file serves as an excellent interview prep and learning resource, and includes:
- Definitions and deep explanations.
- JavaScript implementations (easy to copy-paste and test).
- Simple vs. Industry-level examples.
- Pros, cons, when to use, and when NOT to use.
- Extra interview tips.
